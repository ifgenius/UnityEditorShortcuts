# Unity Editor shortcuts

This package contains useful UnityEditor methods

You can reassign shortcuts in `Edit/Shortcuts`

## Console

Located at `Tools/Console`

* Clear console `(Alt + C)`
* Toggle logs `(Alt + 1)`
* Toggle warnings `(Alt + 2)`
* Toggle errors `(Alt + 3)`
